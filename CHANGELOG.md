# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),

## [0.0.1] - 2026-07-04

### Added
- 🏥 **Symptom Triage Agent** — 症状分诊助手，紧急程度三级分类
- 💊 **Medication Guide Agent** — 用药指导助手，药品查询+相互作用检查
- 📋 **Health Summary Agent** — 健康摘要助手，体征趋势分析
- 🏛️ **Chronic Care Agent** — 慢病管理助手（糖尿病/高血压/COPD/哮喘）
- 🔍 **PGVector RAG** — 医疗知识库向量检索
- 🚀 **Multi-Agent 架构** — 统一路由+工具调用
- 🐳 **Docker Compose** — 一键启动 PostgreSQL(PGVector)+Redis+Ollama
- 📚 **知识种子数据** — 启动时自动加载医疗知识文档
- 🧪 **单元测试** — 基础 API 测试覆盖
- 📖 **完整文档** — README / CONTRIBUTING / requirements
